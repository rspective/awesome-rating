Awesome Rating

v1.0.0

Awesome-rating is a jQuery plugin that allows you to use simple, but flexible rating mechanism. The basic configuration uses Font Awesome, but it's not a problem to replace it with any library you like.

JQuery

Default configuration

The plugin works out of the box. You only need to call .aswesomeRating() jQuery function to have your rating from 1 - 5:

Value: 

$(".jquery-default .rating").awesomeRating({
	valueInitial	: 3,
	targetSelector	: ".jquery-default .result"
});

String values

You are not forced to use numeric values so if you need to have more descriptive ones just pass them as one of the options:

Value: 

$(".jquery-string .rating").awesomeRating({
	valueInitial	: "Awesome",
	values		: [ "Really bad", "Could be worse", "Improvable", "Good", "Awesome" ],
	targetSelector	: ".jquery-string .result"
});

AngularJS

Styling

The plugin allows you to fully customize the styling, not only by changing existing classes but also by allowing you to specify your own that will be applied to specific elements:

Value: 

ratingApp.controller("colourController", [ "$scope", function($scope) {
    $scope.rating = 1;
    $scope.options = {
    	values 			: [ 1, 2, 3 ],
        cssBaseSelected     	: "fa-thumbs-up",
        cssBaseUnselected   	: "fa-thumbs-o-up",
        cssValuesSelected 	: [
            "first-selected",
            "second-selected",
            "third-selected",],
        cssValuesUnselected	: [
            "first-unselected",
            "second-unselected",
            "thirs-unselected",],
        cssHover		: "rating-star-hover-blue"
    };
}]);

Readonly

The plugin is also useful when you want only to display a current value without allowing users to change it. In such case it's recommended to disable the hover behaviour as well.

Value: 

ratingApp.controller("readonlyController", [ "$scope", function($scope) {
	$scope.rating = 3;
	$scope.options = {
		values 		: [ 1, 3, 5, 7, 9 ],
		cssBaseSelected     : "fa-thumbs-up",
		cssBaseUnselected   : "fa-thumbs-o-up",
		readonly 	: true,
		applyHoverCss	: false
	};
}]);

KnockoutJS

Fractional values

There is also a possibility to use fractional values, although the fractional value can only be specified from the code.

Value: 

var modelFractional = {
    rating 	: ko.observable("CD"),
    options 	: {
        values 		: ["A", "B", "C", "D", "E"],
        allowFractional	: true,
        calculateFractional	: function(current, rate) {
            if (current.indexOf(rate) === 1) { return 0.6 };
            return 1;
        },
        allowFractional	: true,
        readonly		: true,
        applyHoverCss	: false
    }
};
ko.applyBindings(modelFractional, $(".knockout-fractional")[0]);

Events

You are also able to attach external event handler to specified by you event that will be fired every time rating value is changed.

Value: 

$(".knockout-events .rating").on("ratingChanged", function(event, value) {
	modelEvents.displayedValue(value);
});

var modelEvents = {
    rating 		: ko.observable(1),
    displayedValue  	: ko.observable(),
    options: {
        htmlEvent	: "mouseenter",
        eventName	: "ratingChanged"
    }
};
ko.applyBindings(modelEvents, $(".knockout-events")[0]);
